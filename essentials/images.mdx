## Afbeelding

### Markdown gebruiken

De [markdown-syntax](https://www.markdownguide.org/basic-syntax/#images) laat je afbeeldingen toevoegen met de volgende code

```md
![titel](/pad/afbeelding.jpg)
```

Houd er rekening mee dat de bestandsgrootte van de afbeelding minder dan 5MB moet zijn. Anders raden we aan om te hosten op een service zoals [Cloudinary](https://cloudinary.com/) of [S3](https://aws.amazon.com/s3/). Je kunt dan die URL gebruiken en embedden.

### Embeds gebruiken

Om meer aanpasbaarheid met afbeeldingen te krijgen, kun je ook [embeds](/writing-content/embed) gebruiken om afbeeldingen toe te voegen

```html
<img height="200" src="/pad/afbeelding.jpg" />
```

### iFrames

Laadt een andere HTML-pagina binnen het document. Meestal gebruikt voor het embedden van video's.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
