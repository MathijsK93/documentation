## Inline code

Om een `woord` of `zin` als code aan te duiden, sluit je het in backticks (`) in.

## Codeblokken

Gebruik [omheinde codeblokken](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) door code in drie backticks te plaatsen en volg de leidende ticks met de programmeertaal van je snippet om syntax highlighting te krijgen. Optioneel kun je ook de naam van je code na de programmeertaal schrijven.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
