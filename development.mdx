<Info>
  **Vereisten**:
  - Node.js versie 19 of hoger
  - Een documentatierepository met een `docs.json` bestand
</Info>

Volg deze stappen om Mintlify op je besturingssysteem te installeren en uit te voeren.

<Steps>
<Step title="Installeer de Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="Preview lokaal">

Navigeer naar je documentatiemap waar je `docs.json` bestand zich bevindt, en voer het volgende commando uit:

```bash
mint dev
```

Een lokale preview van je documentatie is beschikbaar op `http://localhost:3000`.

</Step>
</Steps>

## Aangepaste poorten

Standaard gebruikt Mintlify poort 3000. Je kunt de poort waarop Mintlify draait aanpassen door de `--port` vlag te gebruiken. Om Mintlify bijvoorbeeld op poort 3333 te draaien, gebruik je dit commando:

```bash
mint dev --port 3333
```

Als je probeert Mintlify te draaien op een poort die al in gebruik is, gebruikt het de volgende beschikbare poort:

```md
Poort 3000 is al in gebruik. Probeert 3001 in plaats daarvan.
```

## Mintlify-versies

Houd er rekening mee dat elke CLI-release geassocieerd is met een specifieke versie van Mintlify. Als je lokale preview niet overeenkomt met de productieversie, werk dan de CLI bij:

```bash
npm mint update
```

## Links valideren

De CLI kan helpen met het valideren van links in je documentatie. Om eventuele kapotte links te identificeren, gebruik je het volgende commando:

```bash
mint broken-links
```

## Codeformattering

We raden aan om extensies in je IDE te gebruiken om MDX te herkennen en te formatteren. Als je een VSCode-gebruiker bent, overweeg dan de [MDX VSCode-extensie](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx) voor syntax highlighting, en [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) voor codeformattering.
